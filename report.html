Problem Statement

Two-wheeler breakdowns are common due to overheating, low battery, or high vibrations, especially in remote areas. There is a need for an affordable, real-time monitoring solution for engine health that doesn’t rely on costly OBD systems.

Solution Overview

NetrOn is a smart health guardian system for two-wheelers using an Arduino-powered module that monitors engine temperature, battery voltage, and vibration levels. Data is transmitted via Bluetooth and visualized on a Flask-based dashboard.

Innovation Highlights

  • Low-cost sensor suite (LM35, ADXL345, voltage divider)
  • Threshold-based alert system with LEDs
  • Bluetooth integration for real-time local display
  • Dashboard with interactive visualizations

Key Metrics

Temperature: Normal: 28–32°C | Alert: <60°C

Vibration: Normal: 8–9 units | Alert: <30 units

Battery Voltage: Normal: 7–9V | Alert: <4.5V

Implementation

Built with LM35 (temperature), ADXL345 (vibration), and a voltage divider circuit connected to Arduino Uno. Data is processed and sent via HC-05 Bluetooth to a local device. Web dashboard developed using Flask, HTML, and Tailwind CSS.
Thresholds in the Arduino code trigger LED alerts for heat, vibration, or low battery. The weatherproof, compact system supports future app alerts and OTA updates.
